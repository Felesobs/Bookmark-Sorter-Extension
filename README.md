# 🔖 Bookmark Sorter Extension

เครื่องมือเรียงลำดับบุ๊กมาร์ก (Bookmarks) สำหรับ **Brave**, **Chrome** และเบราว์เซอร์ Chromium อื่น ๆ

ช่วยให้คุณจัดการบุ๊กมาร์กที่รกและไม่เป็นระเบียบให้เรียบร้อยได้อย่างรวดเร็ว

## ✨ คุณสมบัติหลัก

- เรียงลำดับบุ๊กมาร์กตามชื่อ (A-Z / Z-A)
- เรียงตามวันที่เพิ่ม หรือวันที่ใช้งานล่าสุด
- รองรับการเรียงแบบ recursive (รวมโฟลเดอร์ย่อย)
- ปุ่มเรียงลำดับแบบ one-click จาก popup
- ใช้งานง่าย ไม่ต้องติดตั้งซับซ้อน
- รองรับ Manifest V3 (มาตรฐานปัจจุบัน)
